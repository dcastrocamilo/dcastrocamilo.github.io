The blended generalised extreme value distribution

The blended Generalised extreme value (bGEV) distribution is a numerically stable alternative to the usual GEV distribution when the tail parameter \(\xi\) is positive, and it is designed to tackle artificial boundary restrictions inherited by the GEV distribution. It is defined as

\[ H(x \mid \boldsymbol\theta) = F(x \mid \mu_{F}, \sigma_{F}, \xi)^{p(x)} G(x \mid \mu_{G}, \sigma_{G})^{1-p(x)}, \]

where \(\theta=(\mu_F,\sigma_F,\xi,\mu_G,\sigma_G)\), \(F\) is the Fréchet distribution function with location \(\mu_F\), scale \(\sigma_F\) and shape \(\xi\), \(G\) is the Gumbel distribution function with location \(\mu_G\) and scale \(\sigma_G\). The function \(p(x)\) is a weight function defined as

\[ p(x) \equiv p(x\mid a,b,c_1,c_2) = F_{\text{beta}}\left(\frac{x-a}{b-a}\mid c_1, c_2\right), \]

where \(F_{\text{beta}}(\cdot\mid c_1,c_2)\) is the distribution function of a beta distribution with shape parameters \(c_1,c_2>0\)

To provide a more meaningful interpretation of the model’s parameters, we can re-parametrise the bGEV distribution in terms of an \(\alpha\)-quantile and a quantile range (or spread) parameter:

\[ H(x \mid \tilde{\boldsymbol\theta}) = F(x \mid q_{\alpha}, s_{\beta}, \xi)^{p(x)} G(x \mid \tilde{q}_{\alpha}, \tilde{s}_{\beta})^{1-p(x)}, \]

where \(\tilde{\boldsymbol\theta} = (q_\alpha,s_\beta,\xi)\), \(s_\beta = q_{1-\beta/2} - q_{\beta/2}\), and \(q_{\cdot}\) and \(\tilde q_{\cdot}\) are quantiles of the Frechet and the Gumbel distribution, respectively. Under certain mild conditions, the Gumbel parameters are completely determined by those of the Fréchet distribution.

To learn more about the bGEV distribution see the original article.

Simulated example 1

To get familiar with the R-INLA implementation of the bGEV distribution, we start with a simple model where the linear predictor \(\eta\) is linked to the quantile \(q_\alpha\). The model we want to fit is

\[ q_\alpha = \eta(x) = 1+0.4x \]

We start by generating 300 samples from the model

n = 300
x = rnorm(n)
eta.x = 1 + 0.4*x
